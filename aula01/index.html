O que é JavaScrip?

JavaScrip é uma linguagem de programação, voltada para o lado do client-side. Surgiu no ano 1995, criada pelo nome do cara. Também é utilizada na aréa de Back-end, mas suas funções a principio foi criada para o usuario final.Resumindo de forma simples, é o que traz as Interações no site, como por exemplo: Curtir, salvar, compartilhar e muito mais
